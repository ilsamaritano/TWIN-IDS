# Data Directory

This directory includes sanitized sample inputs for running TWIN-IDS.

## Files

- `topology_small_v31.json`: synthetic directed graph with 31 nodes.
- `cve_catalog_sample.csv`: toy vulnerability table with CVSS/EPSS scores.
- `ids_sensor_profiles.csv`: example node-level sensor placement candidates.

## Notes

- Real-world topologies must be anonymized before publication.
- Large artifacts should be hosted externally and linked in `README.md`.
